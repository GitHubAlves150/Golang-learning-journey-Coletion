**Exercício:**

Você for contratoa para criar um sisitema de
análise de testo que processa frases e fornece estátisticas. O sisitema de usar strings, slices e maps para entregar os resultados



## 🎯 Objetivos do Exercício

- Manipular strings (split, trim, lower/upper case)

- Usar slices para armazenar e processar listas

- Utilizar maps para contagens e agregações

- Praticar loops, condicionais e funções
